拡張ルール1

XG conversion rule (tentative)

OSC
XG System Master Tune
/xg/mastertune/ cent [-2400..+2400]
XG Master Volume
/xg/mastervolume/ dB [-100..0]
XG Transpose
/xg/transpose value [-24..+24]
XG System On
/xg/systemon
XG Reverb Type
/xg/reverbtype name
