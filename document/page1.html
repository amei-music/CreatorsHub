MIDI-OSC変換ルール

  • 各種デバイスの接続方法
  • MIDI-OSC変換ルール
  • MIDI-JSON変換ルール
  • OSC-JSON変換ルール

MIDIとOSCの間の変換ルールです。MIDIメッセージがどのようなOSCメッセージに変換されるかを示しています。 （このルールはamei-musicのgithubでオープンに議論・検討して仕様を決めていきます）

Note On [0x9n note_num velocity]
/midi/noteon/ ch note_num velocity
Note Off [0x8n note_num velocity]
/midi/noteoff/ ch note_num velocity
Note Pressure
/midi/notepressure ch note_num value
Control Change
/midi/controlchange ch type value
Program Change
/midi/programchange ch value
