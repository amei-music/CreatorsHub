変換ルール1

Simple examples html page that connects to Creators'Hub and sends simple messages.

MIDI conversion rule (tentative)

OSC
Note On
/midi/noteon/ ch note_num velocity
Note Off
/midi/noteoff/ ch note_num velocity
Not Pressure
/midi/notpressure ch note_num value
Control Change
/midi/controlchange ch type value
Program Change
/midi/programchange ch value
